```
자습ON은 학원 자습실에서 공부하는 학생을 위한 학습 관리 앱입니다. 타이머를 시작하는 순간부터 공부 시간, 쉬는 시간, 과목별 집중 패턴이 자동으로 기록되고, 매일의 목표와 학원 내 랭킹을 한 눈에 확인할 수 있습니다.

■ 공부 시간 자동 기록
타이머로 공부를 시작하면 끝날 때까지 자동으로 측정됩니다. 앱을 닫아도 이어지고, 다음 날 리포트로 돌아옵니다.

■ 실시간 좌석 현황
빈 자리, 공부 중, 휴식 중 상태를 실시간으로 확인하고 원하는 자리에 바로 착석하세요. 학원 관리자가 배정한 좌석 정보도 앱에서 확인할 수 있습니다.

■ 오늘의 계획 · 추천 학습량
학원 운영 데이터와 최근 학습 흐름을 기반으로, 오늘 집중할 과목과 분량을 앱이 직접 제안합니다. 한 번의 탭으로 오늘 일정에 추가할 수 있습니다.

■ 주간 · 월간 리포트
공부 시간, 목표 달성률, 과목 밸런스를 차트로 확인하세요. 부모님과 선생님도 학원 관리자 페이지에서 같은 데이터를 볼 수 있어 소통이 쉬워집니다.

■ 학원 내 랭킹
같은 공간에서 공부하는 친구들과 건강하게 경쟁하세요. 일간·주간 순위로 자연스럽게 학습 동기를 유지합니다.

■ 필요한 알림만, 조용히
입실·퇴실, 목표 달성, 휴식 타이밍 등 꼭 필요한 순간에만 알립니다. 광고 알림은 없습니다.

■ 학생 데이터를 최소한만 수집합니다
자습ON은 학습 관리에 꼭 필요한 정보만 수집하며, 외부 마케팅 네트워크에 공유하지 않습니다. 모든 통신은 TLS로 암호화됩니다.

자습ON을 학원에 도입하고 싶으신 운영자는 hyphendev2025@gmail.com으로 문의 주세요.
```

## 5. 주의 / 리스크

- **수학 하드코드**: 회원가입 문구 "수학학원 학습 관리", 일부 화면에서 과목이 '수학'으로 고정. 심사에서 카테고리 오기/오류 주장 받을 가능성 낮지만, 수학 외 학원 확장 시 문구 갱신 필요.
- **관리자 접근**: long-press 로고로 관리자 로그인 토글. Apple 리뷰어가 발견 못 할 수 있음 → 심사 노트에 **명시 필수**.
- **체크인 제스처**: 스와이프-업 필수. VoiceOver 사용자 접근성 이슈 (QA 리포트 C1 참조). 장기 개선 권장.
- **학번 자동 발급**: 회원가입 시 타임스탬프 기반 학번 자동 배정. 실학번과 매칭은 관리자가 수동. 심사 영향 없음.
